# Triage labels

| Role | Label |
|------|-------|
| Needs triage | `needs-triage` |
| Needs info | `needs-info` |
| Ready for agent | `ready-for-agent` |
| Ready for human | `ready-for-human` |
| Won't fix | `wontfix` |

These labels should exist in the GitHub repo. The `triage` skill will apply them during issue state transitions.
